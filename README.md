# BlazorTemplate

A project template that combines multiple Blazor patterns and enables them to share a common UI

- Blazor WebAssembly
- Blazor Server
- Maui
- Winforms
- WPF
- Photino

## Install template

```shell
dotnet new install Yu-Core.Template
```

## 

## Uninstall template

```shell
dotnet new uninstall Yu-Core.Template
```
