# BlazorTemplate

A Blazor template with Masa.Blazor component library integrated, supporting multiple platforms including Blazor Server, Blazor WebAssembly, Blazor Hybrid.

- Blazor WebAssembly
- Blazor WebApp (Global Server) 
- Gtk
- Maui
- Photino
- Winforms
- WPF

## Install template

```shell
dotnet new install Yu-Core.BlazorTemplate
```

## 

## Uninstall template

```shell
dotnet new uninstall Yu-Core.BlazorTemplate
```
